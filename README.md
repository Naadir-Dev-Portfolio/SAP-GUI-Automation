# SAP-GUI-Automation

SAP GUI automation using VBA — standard templates and scripted workflows

Built by [Naadir](https://github.com/Naadir-Dev-Portfolio)

## Overview

VBA-based automation framework for SAP GUI scripting. Includes standard templates for common SAP transactions and workflows, reducing manual data entry and improving process efficiency.

## Features

- SAP GUI scripting with VBA
- Standard transaction templates
- Automated data input workflows
- Session and connection management
- Error handling and logging
- Integration with Excel for data exchange

## Tech Stack

VBA · SAP GUI Scripting · Excel

## Usage

1. Enable SAP Scripting in SAP Logon configuration
2. Load VBA scripts into Excel or SAP VB Editor
3. Configure transaction codes and data mappings
4. Execute macros to automate workflows

## Notes

Requires SAP GUI client and proper user permissions for target transactions.
